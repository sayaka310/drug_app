## 概要
* Laravel Breezeをベースに実装
* User以外にモデルを2つ以上でリレーション
* CRUD機能(登録、更新、照会、一覧、削除)を作成
* 画像ファイルの複数同時アップロードと表示(Firebase Storage) 表示方法は複数表示
* SNS認証(github,google)
* Bootstrap・TailwindCSS等を使用
